# 功能特性

探索 NimoteCode 提供的所有强大功能。

## 核心编辑器

### 语法高亮

NimoteCode 支持 50+ 种编程语言的语法高亮：

| 类别 | 语言 |
|------|------|
| **Web** | JavaScript、TypeScript、HTML、CSS、Vue、React |
| **后端** | Python、Java、Go、Rust、C/C++、C# |
| **移动** | Swift、Kotlin、Dart、Flutter |
| **数据** | SQL、JSON、YAML、TOML、XML |
| **脚本** | Bash、PowerShell、Python、Ruby |
| **配置** | Dockerfile、Kubernetes、Nginx、Apache |

### 代码折叠

折叠和展开代码块以便更好地导航：

- 函数和方法
- 类和结构体
- 条件块
- 循环
- 注释

### 编辑器功能

| 功能 | 描述 |
|------|------|
| **自动保存** | 修改后 5 秒自动保存 |
| **撤销/重做** | 多级撤销历史 |
| **剪切/复制/粘贴** | 完整剪贴板支持 |
| **全选** | 快速选择所有内容 |
| **光标位置** | 实时显示行号和列号 |
| **上下文菜单** | 右键快速操作 |
| **键盘快捷键** | Ctrl+C/V/X 等 |
| **虚拟键盘** | 移动端优化的编程键盘 |

### 大文件支持

- 支持最多 **100,000 行**的文件
- 大文件分块读取
- 自动截断警告
- 流畅的滚动性能

### 保存状态

- **脏标记**: 显示未保存的更改
- **保存指示器**: 显示正在保存中
- **保存确认**: 保存完成时的视觉反馈

## SSH 远程开发

### 连接管理

| 功能 | 详情 |
|------|------|
| **认证方式** | 密码和 SSH 密钥支持 |
| **超时** | 10 秒连接超时 |
| **自动重连** | 最多 2 次重连尝试 |
| **状态显示** | 实时连接状态 |

### 远程文件浏览器

- **树形视图**: 可视化文件系统导航
- **文件图标**: 类型特定的图标
- **路径导航**: 自动展开到当前文件
- **目录缓存**: 性能优化
- **刷新**: 手动目录刷新
- **折叠全部**: 快速折叠所有展开的文件夹

### 文件操作

| 操作 | 描述 |
|------|------|
| **创建** | 新建文件和文件夹 |
| **编辑** | 实时编辑并自动保存 |
| **重命名** | 重命名文件和目录 |
| **删除** | 删除文件和目录（带确认） |
| **打开** | 在编辑器中打开文件 |

### 远程文件编辑

- 在内置编辑器中实时编辑远程文件
- 修改后 5 秒自动保存
- 大文件分块读取
- 超过 100,000 行的文件会被截断并显示警告
- UTF-8 编码支持

### 多终端会话

- **6 个并发会话**: 同时运行最多 6 个终端
- **会话持久化**: 切换面板时会话保持
- **独立通道**: 每个终端有唯一通道 ID
- **自动清理**: 关闭工作区时清理所有会话

### AI 终端分析（新功能！）

NimoteCode 可以使用 AI 分析终端输出：

- **自动分析**: AI 自动分析命令输出
- **错误诊断**: 识别错误并提供修复建议
- **摘要生成**: 提取关键信息
- **下一步建议**: 推荐后续操作
- **剪贴板集成**: 分析提示自动复制到剪贴板

**分析格式：**

```
**快速摘要:** 发生内容的简要概述

**错误和警告:** 检测到的问题列表及修复建议

**输出摘要:** 从输出中提取的关键信息

**下一步:** 推荐的后续操作
```

## AI 助手

### 主要功能

| 功能 | 描述 |
|------|------|

### 配置管理

- **自定义端点**: 支持自定义 API 端点

- **文件标记**: 显示修改、新增、删除状态图标

### 文件历史

- **所有版本**: 查看所有历史版本

### 行历史

- **行信息**: 显示特定行的修改历史
- **最后修改**: 显示最后修改该行的提交
- **作者信息**: 显示最后修改的作者
- **修改时间**: 显示最后修改的时间
