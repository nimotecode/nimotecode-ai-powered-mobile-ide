# 功能特性

探索 NimoteCode 提供的所有强大功能。

## 核心编辑器

### 语法高亮

NimoteCode 支持 50 多种编程语言的语法高亮：

| 分类 | 语言 |
|------|------|
| **Web** | JavaScript, TypeScript, HTML, CSS, Vue, React, Svelte, Next.js |
| **后端** | Python, Java, Go, Rust, C/C++, C#, PHP, Ruby, Node.js |
| **移动端** | Swift, Kotlin, Dart, Flutter, React Native |
| **数据** | SQL, JSON, YAML, TOML, XML, CSV, Protobuf |
| **脚本** | Bash, PowerShell, Python, Ruby, Perl |
| **配置** | Dockerfile, Kubernetes, Nginx, Apache, AWS CloudFormation |

### 代码折叠

折叠和展开代码块，提高导航效率：

- 函数和方法
- 类和结构体
- 条件块（if/else, switch/case）
- 循环（for, while, do-while）
- 注释和文档
- 导入语句和命名空间

### 编辑器功能

| 功能 | 描述 |
|------|------|
| **自动保存** | 5 秒后自动保存更改 |
| **撤销/重做** | 多级撤销历史 |
| **剪切/复制/粘贴** | 完整的剪贴板支持和键盘快捷键 |
| **全选** | 快速选择所有内容 |
| **光标位置** | 实时显示行和列 |
| **右键菜单** | 右键快速操作 |
| **键盘快捷键** | Ctrl+C/V/X, Cmd+S 等 |
| **虚拟键盘** | 移动端优化的编程键盘 |
| **自动补全** | 智能代码补全建议 |
| **代码检查** | 实时错误和警告检测 |

### 大文件支持

- 高效处理多达 **100,000 行**的文件
- 大文件的分块读取
- 对超大文件的自动截断警告
- 60fps 动画的流畅滚动性能
- 内存高效的文件处理

### 保存状态

- **脏标记**：未保存更改的视觉指示器
- **保存指示器**：保存操作的进度指示器
- **保存确认**：保存完成时的视觉反馈
- **自动保存计时器**：可配置的自动保存延迟

## SSH 远程开发

### 连接管理

| 功能 | 详情 |
|------|------|
| **认证方式** | 密码和 SSH 密钥支持（RSA, ED25519, ECDSA） |
| **超时** | 可配置的连接超时（默认 10 秒） |
| **自动重连** | 带指数退避的自动重连 |
| **状态显示** | 带视觉指示器的实时连接状态 |
| **连接池** | 多个同时连接 |
| **会话管理** | 保存和管理多个服务器配置文件 |

### 远程文件浏览器

- **树形视图**：可展开/折叠的可视化文件系统导航
- **文件图标**：不同类型文件的特定图标
- **路径导航**：面包屑导航和路径搜索
- **目录缓存**：智能缓存的性能优化
- **刷新**：手动和自动目录刷新
- **折叠全部**：快速折叠所有文件夹
- **搜索**：按名称查找文件和目录
- **过滤**：按类型或扩展名过滤文件

### 文件操作

| 操作 | 描述 |
|------|------|
| **创建** | 创建新文件和目录 |
| **编辑** | 实时编辑，自动保存 |
| **重命名** | 重命名文件和目录 |
| **删除** | 删除文件和目录（带确认） |
| **复制** | 在服务器内复制文件和目录 |
| **移动** | 在不同目录间移动文件 |
| **上传** | 将本地文件上传到远程服务器 |
| **下载** | 下载远程文件到本地设备 |
| **权限** | 查看和修改文件权限 |
| **符号链接** | 查看和管理符号链接 |

### 终端功能

- **Shell 支持**：Bash、Zsh、Fish 和自定义 Shell
- **Tab 补全**：智能命令补全
- **命令历史**：跨会话持久化命令历史
- **多终端**：单个会话中的多个终端标签
- **分屏终端**：多个命令的分屏终端视图
- **复制/粘贴**：支持键盘快捷键的轻松复制/粘贴
- **搜索**：搜索终端输出
- **导出**：将终端会话导出到文件

## 高级功能

### 可视化 Git 集成

- **仓库状态**：修改、暂存和未跟踪文件的概览
- **提交历史**：提交历史的视觉表示
- **差异查看器**：并排差异比较
- **按文件跟踪**：按文件和行查看更改
- **分支管理**：创建、切换和管理分支
- **暂存区**：可视化的暂存区管理
- **远程仓库**：连接到 GitHub、GitLab、Bitbucket

### AI 代码智能

- **上下文感知补全**：基于上下文的智能代码补全
- **重构建议**：自动化重构建议
- **代码解释**：获取复杂代码的解释
- **错误检测**：AI 驱动的错误检测
- **性能建议**：优化代码性能
- **语言支持**：支持 50+ 编程语言的 AI
- **自定义模型**：连接您喜欢的 LLM（OpenAI、Claude 等）

### 终端自定义

- **主题支持**：多种终端配色主题
- **字体自定义**：自定义字体设置
- **键盘快捷键**：可自定义的键盘快捷键
- **配置文件管理**：保存多个终端配置文件
- **自动调整大小**：终端自动调整到内容大小
- **滚屏缓冲**：可配置的滚屏缓冲区大小

### SFTP 文件浏览器

- **拖放**：在本地和远程之间拖放文件
- **远程浏览器**：完整的远程文件系统导航
- **文件预览**：打开前预览文件
- **批量操作**：对多个文件执行操作
- **连接安全**：端到端加密连接
- **传输队列**：管理多个文件传输
- **恢复传输**：恢复中断的文件传输

### 云同步与备份

- **安全同步**：端到端加密云同步
- **设置备份**：备份和恢复应用程序设置
- **项目同步**：在设备间同步项目
- **自动备份**：每日自动备份
- **云存储**：支持 iCloud、Google Drive、Dropbox
