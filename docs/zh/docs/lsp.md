# LSP 面板

LSP 面板帮助你更早发现问题并更快完成修复，适合远程项目的持续开发。

## 核心价值

- 在同一面板集中查看诊断
- 从问题项一键跳转源码位置
- 快速应用代码动作，降低手工修复成本
- 通过安装引导完成语言服务能力准备

## 你可以完成的操作

- 查看并筛选错误/警告
- 快速打开问题对应文件位置
- 刷新并应用可用代码动作
- 使用安装向导配置支持语言服务
- 无自动修复时回退到 AI 修复流

## 推荐使用路径

1. 运行代码后打开 LSP 面板。
2. 优先处理错误级别问题。
3. 应用可用的代码动作。
4. 复查诊断并迭代。
5. 复杂问题交给 Debug 或 AI。

## Pro 说明

LSP 属于 Pro 工作流，并针对 SSH 远程工作区深度优化。
