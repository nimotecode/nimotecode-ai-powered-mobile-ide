# Source Control

Source Control 将完整 Git 工作流带到 NimoteCode，支持在移动端完成审查、提交与同步。

## 你可以完成的能力

- 查看已暂存、未暂存、冲突文件
- 按文件或批量暂存/丢弃改动
- 执行 commit、pull、push、fetch
- 管理分支与 stash 生命周期
- 查看提交/文件/行级历史与差异

## Git AI 辅助

内置 AI 可帮助你：

- 生成提交信息
- 解释改动内容
- 提示改动风险

## 典型提交流程

1. 先看差异内容。
2. 暂存目标改动。
3. 生成或编写提交信息。
4. 提交并同步。
5. 在目标分支验证结果。

## 团队价值

- 移动端交付效率更高
- 紧急修复路径更短
- AI 辅助提升改动质量
