# SSH 工作区

SSH 工作区把 NimoteCode 变成移动端远程开发控制台。

## 你可以完成什么

- 保存并复用服务器连接配置
- 使用密码或私钥方式连接
- 在远程项目目录中直接开发
- 远程浏览、编辑、创建、重命名、删除文件
- 让终端与编辑器保持同一工作区上下文

## 典型远程会话

1. 选择 SSH 配置档。
2. 连接并打开工作区根路径。
3. 编辑文件并执行终端命令。
4. 用 Source Control 与 Debug/LSP 完成验证与交付。

## 推荐实践

- 按环境拆分配置档
- 关键环境优先使用私钥认证
- 明确工作区根路径，减少误操作风险
