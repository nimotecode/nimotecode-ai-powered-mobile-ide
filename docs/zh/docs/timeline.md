# Timeline 面板

Timeline 将运行信号转化为可执行的排障叙事，帮助你更快完成定位和修复。

## Timeline 提供什么

- Trace 视图：因果链与根因分析
- Event 视图：按时间顺序查看关键事件
- 严重级别与来源筛选
- 根因提示与修复建议
- 错误/警告趋势计数

## 核心价值

- 更快理解事故全貌
- 把症状与根因准确对齐
- 为团队协作提供清晰交接上下文

## 典型排障流程

1. 先筛选高严重级别事件。
2. 打开相关 Trace。
3. 查看可能根因。
4. 执行修复建议或转交 AI。
5. 用新事件流验证修复效果。

## 跨面板协同价值

Timeline 与 LSP、Debug、AI、Tasks 联动时效果最佳，可形成“发生了什么 + 下一步做什么”的统一视图。
