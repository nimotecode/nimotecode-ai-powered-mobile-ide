## 入门指南

### 如何连接远程服务器？

1. 打开应用并进入 SSH 面板
2. 点击 "+" 按钮添加新连接
3. 输入服务器详情：
   - **主机**：服务器 IP 或域名
   - **端口**：22（默认 SSH 端口）
   - **用户名**：服务器用户名
   - **认证方式**：选择密码或 SSH 密钥
4. 保存并连接！

### 支持哪些认证方式？

- **密码认证**：简单的用户名和密码登录
- **SSH 密钥认证**：更安全的密钥认证方式

### 可以离线使用吗？

部分功能支持离线使用（如代码编辑器），但远程开发功能需要网络连接。

## 技术问题

### 系统要求是什么？

**Android：**
- Android 8.0 或更高版本
- 最低 2GB 内存（推荐 4GB）
- ARM64 或 x86_64 处理器

**iOS：**
- iOS 14.0 或更高版本
- iPhone、iPad 或 iPod Touch
- 最低 2GB 内存

### 数据安全吗？

是的！我们采用：
- Firebase 身份验证
- 加密 SSH 连接
- 安全令牌存储
- 无数据保留政策

### SSH 凭据存储在哪里？

所有凭据仅在**本地加密存储**，不会上传到任何服务器。

## 故障排除

### SSH 连接失败怎么办？

1. **检查网络**：确保设备可以访问服务器
2. **验证凭据**：确认用户名和密码/密钥正确
3. **检查端口**：确认 SSH 端口（默认 22）已开放
4. **查看日志**：检查应用内的错误信息

### 终端显示乱码？

这通常是字符编码问题：
- 确保服务器使用 UTF-8 编码
- 检查终端字体是否支持您的字符集

### AI 助手无响应？

1. 检查网络连接
2. 验证 API 密钥是否正确
3. 确认 API 端点可访问
4. 尝试切换到其他 AI 提供商

### 应用崩溃怎么办？

1. 重启应用
2. 清除应用缓存
3. 更新到最新版本
4. 如问题持续，请联系支持
