## 安装指南

### Android

1. **从 Play 商店安装：**
   - 打开 Google Play 商店
   - 搜索 "NimoteCode"
   - 点击"安装"
   - 打开应用

2. **从 APK 安装：**
   - 从 GitHub releases 下载 APK
   - 在设置中启用"安装未知来源应用"
   - 打开 APK 文件
   - 点击"安装"
   - 打开应用

### iOS

1. 打开 App Store
2. 搜索 "NimoteCode"
3. 点击"获取"
4. 打开应用

## 首次启动

首次打开 NimoteCode 时：

1. **登录**：创建账户或使用 Google/Apple 登录
2. **选择计划**：选择免费版或专业版
3. **添加连接**：设置您的第一个 SSH 或 GitHub 连接
4. **开始编码**：打开文件并开始编辑！
