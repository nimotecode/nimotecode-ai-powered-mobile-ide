# 简介

欢迎使用 NimoteCode，一款专为 Android 和 iOS 设计的强大移动端远程开发 IDE。

## 什么是 NimoteCode？

NimoteCode 是一款跨平台移动集成开发环境 (IDE)，将桌面开发环境的完整功能带到您的移动设备上。无论您使用手机还是平板，NimoteCode 都能提供流畅的编码体验。

## 主要特性

### 🖥️ 现代代码编辑器

- **语法高亮**: 支持 50+ 编程语言
- **代码折叠**: 折叠和展开函数、类和代码块
- **自动保存**: 修改后 5 秒自动保存
- **大文件支持**: 支持最多 100,000 行的文件
- **撤销/重做**: 多级操作历史
- **虚拟键盘**: 移动端优化的编程键盘

### 🔐 安全 SSH 远程开发

- **SSH 连接**: 支持密码和密钥认证
- **SFTP 文件管理器**: 浏览、上传、下载和编辑远程文件
- **多终端**: 同时运行最多 6 个终端会话
- **自动重连**: 断线后自动重连（最多 2 次）
- **连接状态**: 实时显示连接状态

### 🖥️ 终端功能

- **xterm 兼容**: 完整终端模拟器
- **ANSI 颜色**: 完整颜色支持
- **快捷命令**: 一键执行常用命令
- **AI 分析**: 智能终端输出分析
- **全屏模式**: 终端支持全屏显示

### 🤖 AI 智能助手

- **多提供商支持**: OpenAI、Ollama、SiliconFlow、Groq
- **聊天模式**: 提问并获得帮助
- **编辑模式**: AI 辅助代码修改
- **上下文感知**: 自动包含当前文件上下文
- **流式响应**: 实时显示 AI 响应

### 📦 GitHub 集成

- **仓库访问**: 浏览所有仓库（公开和私有）
- **文件编辑**: 直接从设备编辑文件
- **自动提交**: 更改自动提交
- **分支支持**: 使用不同分支工作
- **OAuth 认证**: 安全的 GitHub 登录

### 🔧 Git 版本控制

- **可视化 Git**: 一目了然查看更改
- **提交历史**: 浏览和比较提交
- **差异查看器**: 可视化比较更改
- **行历史**: 查看谁修改了什么
- **实时状态**: 每 8 秒自动刷新状态

### 🔍 强大搜索

- **文件搜索**: 按名称查找文件
- **内容搜索**: 在文件内搜索
- **符号搜索**: 跳转到函数、类、变量
- **远程搜索**: 在远程服务器搜索

### 🎨 可定制界面

- **6 种主题**: Dark、Light、Solarized Dark、Solarized Light、Dracula、Monokai
- **响应式布局**: 适配手机和平板
- **三栏布局**: 高效的工作区组织
- **状态栏**: 显示光标位置、编码、保存状态

## 技术架构

NimoteCode 采用现代技术构建：

| 组件 | 技术 |
|------|------|
| **前端** | Flutter (Dart) |
| **后端** | Rust (通过 Flutter Rust Bridge) |
| **状态管理** | Provider |
| **本地存储** | Hive |
| **HTTP 客户端** | Dio |

### 为什么选择 Rust？

Rust 提供：
- **性能**: SSH 和 Git 操作接近原生速度
- **安全**: 关键操作的内存安全代码
- **可靠性**: 强大的错误处理

### 关键插件

- **SSH 插件**: SFTP 文件操作和 SSH 终端
- **Git 插件**: 远程 Git 命令执行
- **GitHub 插件**: GitHub API 调用
- **高亮插件**: 代码语法高亮
- **差异插件**: 文件差异计算
- **代码分析插件**: 符号识别和分析
