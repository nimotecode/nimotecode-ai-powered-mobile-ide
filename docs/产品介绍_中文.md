# NimoteCode 产品介绍

【一句话产品定位】
随身携带的 AI 编程工作台，让你在手机上就能完成从编码到部署的完整开发工作

【解决什么问题】
- 想改代码但电脑不在身边
- 远程服务器维护还要到处找工具
- 遇到问题半天查不出原因
- 复杂操作记不住命令
- 开发过程零散，出问题难追溯
- 多工具切换太麻烦，效率低
- 想利用碎片时间学习但环境受限

【典型使用场景】
1. **通勤路上的紧急修复**  
   地铁上收到告警，掏出手机连接服务器，几分钟就定位并修复了问题

2. **咖啡店里的快速原型**  
   灵感来了，用 AI 助手直接生成代码框架，边写边在终端里运行验证

3. **沙发上的服务器维护**  
   躺在家里就能查看日志、更新配置、部署代码，不用开电脑

4. **团队协作的及时响应**  
   同事提交了代码，随时查看差异、合并分支，不耽误项目进度

5. **问题复盘与持续学习**  
   通过时间线回看整个开发过程，AI 帮你分析哪里出了问题，下次怎么避免

【你可以做什么】
- 随时随地写代码，支持多种编程语言高亮
- 同时打开多个文件，在它们之间轻松切换
- 一键连接远程服务器，直接编辑文件、运行命令
- 同时打开多个终端会话，并行处理多个任务
- 保存常用命令，一键快速执行
- 在终端输出中快速搜索查找内容
- 可视化管理代码版本，查看修改、提交代码
- 查看文件和代码行的修改历史，了解谁在什么时候改了什么
- 多人协作时帮助解决代码冲突
- 让 AI 帮你写代码、解释代码、甚至自动修改文件
- 在代码中搜索特定文本内容，快速定位
- 按文件名快速查找文件
- 搜索函数、变量、类等代码符号
- 查看代码结构，快速导航到函数或类
- 管理开发任务，记录完成状态
- 记录整个开发过程，通过时间线回看
- AI 帮你分析开发过程中的问题
- 调试代码，设置断点查看变量
- 同步本地和远程的项目文件
- 管理多个 SSH 连接，安全保存登录信息
- 切换不同的界面主题，选择自己喜欢的风格
- 配置键盘相关的偏好选项
- 选择不同的 AI 模型，定制 AI 体验
- 保存和使用代码片段，快速插入常用代码
- 快速登录账号，支持多种登录方式
- 接收应用通知，及时了解重要信息

【为什么选择我们】
- **真·移动端优先**：专为手机和平板设计，操作流畅不将就
- **AI 不只是聊天**：能真正帮你执行操作，从「问答」到「做事」
- **远程开发零门槛**：SSH、终端、Git 一体化，一个应用全搞定
- **过程可追溯**：时间线记录开发全程，问题定位快人一步
- **功能超全面**：从代码编辑、调试、版本控制到任务管理，应有尽有
- **多终端并行**：同时运行多个终端，效率翻倍
- **智能搜索**：文件、内容、符号都能搜，快速找到想要的
- **历史可追踪**：文件、代码行的修改历史一目了然
- **个性化定制**：主题、键盘、AI 模型都能按自己喜好设置
- **跨设备无缝**：手机、平板都能用，场景切换不中断

【适合谁使用】
- 需要随时处理线上问题的后端工程师
- 喜欢在移动中构思和验证想法的全栈开发者
- 经常需要维护远程服务器的运维人员
- 希望利用碎片时间学习编程的学生
- 追求高效开发体验的技术极客
- 需要随时随地响应团队协作的开发人员
- 想要完整记录和复盘开发过程的学习者
