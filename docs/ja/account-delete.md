# アカウント削除

アカウントとすべてのデータを削除する方法は以下の通りです。

## アプリ内での削除

1. NimoteCodeアプリを開きます
2. **設定** → **アカウント**に移動します
3. **アカウントを削除**をタップします
4. 確認の指示に従います

## データの削除

アカウントを削除すると、以下が永久に削除されます：
- アカウント情報
- サーバープロファイル
- アプリ設定
- チャット履歴
- AIエージェントデータ

## リクエストによる削除

アプリにアクセスできない場合は、サポートまでご連絡ください。確認後、手動で削除を処理します。
