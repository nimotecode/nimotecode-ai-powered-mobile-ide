# FAQ

## NimoteCodeはリモート開発専用ですか？

いいえ。Local ExplorerモードまたはRemote Explorer（SSH）モードで作業できます。

## どのAIプロバイダーがサポートされていますか？

組み込みテンプレートには、OpenAI、Anthropic、Gemini、Grok、DeepSeek、Zhipu、Moonshot、Ollama、カスタムOpenAI互換エンドポイントが含まれています。

## Source Controlで何ができますか？

ステージング/アンステージング/破棄、コミット、プッシュ/プル/フェッチ、ブランチ/stashの管理、履歴の表示、Git AI分析ツールの使用ができます。

## プロジェクトタスクを実行できますか？

はい。Tasksパネルを使用してタスクを作成して実行し、ターミナルで出力を検査できます。

## Timelineは何に使用されますか？

Timelineは、トラブルシューティングのための因果分析フローを含む、イベントとトレースを検査するのに役立ちます。

## なぜ機能がロックされているのですか？

一部の機能はProゲートされています。サブスクリプション状態は、アプリのプレミアムアクセスフローによって検証されます。
