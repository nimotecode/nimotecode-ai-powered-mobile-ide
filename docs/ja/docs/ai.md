# AIアシスタント

NimoteCodeには2つのAI体験が含まれています：直接支援のためのAIチャットと、多段階実行のためのAIエージェントです。

## AIチャット

コーディング中に迅速な回答が必要なときにAIチャットを使用します：

- なじみのないコードを説明する
- ターミナルエラーを分析する
- リファクター計画を提案する
- コミット前により安全な次のステップを生成する

## AIエージェント

作業に複数のアクションとチェックが必要なときにAIエージェントを使用します：

- ステップバイステップの実行フロー
- 進捗の可視化と実行状況
- 機密アクションの前のリスクプロンプト
- 制御の好みに基づいたオートモードまたは確認モード

## プロバイダーの柔軟性

NimoteCodeは複数のプロバイダーテンプレートをサポートしています：

- OpenAI
- Anthropic
- Gemini
- Grok
- DeepSeek
- Zhipu
- Moonshot
- Ollama
- カスタムOpenAI互換エンドポイント

## 設定できること

- モデルIDとエンドポイント
- サンプリング動作と出力サイズ
- ストリーミング応答モード
- システムプロンプトとコンテキストサイズ
- コードベース認識応答のためのファイルコンテキスト使用

## ベストプラクティスの設定

1. 1つのプライマリクラウドモデルと1つのバックアップモデルを保持します。
2. 決定論的なコードタスクには低い創造性を使用します。
3. プロジェクトレベルの分析のためにファイルコンテキストをオンにします。
4. 本番で実行する前にAI出力を確認します。
