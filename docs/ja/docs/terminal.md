# ターミナル

ターミナルは、NimoteCodeにおけるビルド、テスト、デプロイ、インシデント操作のコマンドセンターです。

## コア機能

- マルチタブターミナルセッション
- リアルタイムコマンド出力
- 高速なログターゲッティングのための出力検索
- 繰り返しアクションのためのクイックコマンドショートカット
- 分析のために主要なターミナル出力をAIに送信
- 長いワークフローのためにタブを名前変更して整理

## 典型的なコマンドジャーニー

1. 現在のワークスペースでターミナルを開きます。
2. build/test/deployコマンドを実行します。
3. エラーと警告の出力を検索します。
4. 複雑なログをAIまたはTimelineにエスカレーションします。

## なぜ重要なのか

- モバイルでのコマンドから結果までのループがより速くなる
- リモートでインシデントを処理する際の可視性が向上
- エディター、ソースコントロール、AIパネルとの連続性が強化
