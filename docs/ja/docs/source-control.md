# Source Control

Source Controlは、モバイルから変更をレビュー、コミット、同期できるように、完全なGitワークフローをNimoteCodeにもたらします。

## できること

- ステージング済み、未ステージング、競合ファイルをレビューする
- 一括またはファイルごとに変更をステージングまたは破棄する
- pull/push/fetchでコミットして同期する
- ブランチとstashワークフローを管理する
- コミット/ファイル/行の履歴と差分を検査する

## Git AI支援

組み込みAIが以下を支援できます：

- コミットメッセージを生成する
- 変更されたファイルを説明する
- 潜在的な変更リスクを強調表示する

## 典型的なコミットフロー

1. ファイルの差分をレビューします。
2. 選択した変更をステージングします。
3. コミットメッセージを生成または記述します。
4. コミットして同期します。
5. ターゲットブランチで検証します。

## チームが使用する理由

- モバイルでのより速いコードデリバリー
- 緊急修正のための摩擦の低減
- AI支援レビューによるより良い変更品質
