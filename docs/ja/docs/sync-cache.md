# Sync / Cacheパネル

Sync / Cacheは、明確な方向制御でローカルとリモートワークスペース間でプロジェクトコンテンツを移動するためのProパネルです。

## 2つの転送モード

- Sync：ローカルプロジェクトからリモートワークスペースへ
- Cache：リモートワークスペースからローカルプロジェクトへ

## 得られるもの

- ワークスペースコンテキストに基づいたモード認識のプライマリアクション
- アクティブなワークスペース境界に制約されたターゲットパス選択
- 状況とパス記録を含む転送履歴
- 実行前と実行中の明確な状態可視化

## 典型的な使用例

- ローカル編集をリモートランタイム環境にプッシュする
- オフラインレビューのためにリモートプロジェクト状態をプルする
- サーバーサイド更新のポータブルキャッシュを保持する

## なぜ重要なのか

- より速い環境引き継ぎ
- ローカルとリモートコピー間のより良い一貫性
- 明示的な方向とパス制御によるより安全な転送ワークフロー
