# LSPパネル

LSPパネルは、リモートプロジェクトで作業中に問題を早期に発見し、修正をより速く適用するのに役立ちます。

## コアバリュー

- 1つの場所に診断を表示する
- 問題からソース行に直接ジャンプする
- 手動ステップを少なくしてコードアクションを適用する
- ガイド付きフローで言語サーバーサポートをインストールして検証する

## できること

- エラーと警告を表示してフィルターする
- エディターで問題の場所を即座に開く
- 利用可能なコードアクションを更新して適用する
- サポートされている言語サーバーのインストーラーガイダンスを使用する
- 自動アクションが利用できない場合はAI修正フローにフォールバックする

## ベスト使用パターン

1. コードを実行してLSPパネルを開きます。
2. 最初にエラーにフィルターします。
3. 利用可能な場合はクイックフィックスを適用します。
4. 診断を再確認して繰り返します。
5. 複雑なケースをDebugまたはAIに引き継ぎます。

## Proノート

LSPはProワークフローとして提供され、アクティブなSSHリモートワークスペース使用向けに最適化されています。
