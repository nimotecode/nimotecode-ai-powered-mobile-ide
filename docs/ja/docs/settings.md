# 設定

設定により、NimoteCodeを安定した長時間セッションのモバイル開発環境に形成できます。

## アカウントとサブスクリプション

- サインインとアカウントID管理
- サブスクリプション状況とトライアル可視化
- サブスクリプション管理と購入復元エントリー

## 外観と言語

- テーマモード制御
- IDEテーマ選択
- 多言語切り替え

## エディター体験

- フォントサイズと行間隔の調整
- フォントファミリーの好み
- セッション間で永続的なエディタースタイル

## キーボードと入力

- ソフトキーボードの有効/無効
- 触覚とサウンドフィードバックオプション
- 数字行と機能行の切り替え
- ジェスチャーサポートと機能キーマッピング
- キーボードテーマとリセットオプション

## ポリシーアクセス

設定には以下への直接エントリーが含まれています：

- [アカウント削除](/ja/account-delete)
- [プライバシーポリシー](/ja/privacy)
- [利用規約](/ja/terms)

## 推奨される最初の設定

1. 言語とテーマを設定します。
2. エディターの文字組版を調整します。
3. デバイス用にキーボード動作を設定します。
4. サブスクリプションとアカウント状態を確認します。
