# Timelineパネル

Timelineは、ランタイム信号を実行可能なトラブルシューティングナラティブに変換します。

## Timelineが提供するもの

- 因果分析のためのトレースビュー
- 時系列検査のためのイベントリストビュー
- 高速な絞り込みのための重大度とソースフィルター
- 根本原因のヒントと修正提案
- エラー/警告トレンド認識のための状況カウンター

## なぜ重要なのか

- より速いインシデント理解
- 症状と根本原因の間のより良いアライメント
- チームコラボレーションのための明確な引き継ぎコンテキスト

## 典型的なトラブルシューティングフロー

1. 重大度の高いイベントにフィルターします。
2. 関連するトレースを開きます。
3. 考えられる根本原因をレビューします。
4. 修正提案を適用するか、AIにエスカレーションします。
5. 新しいイベントストリームで結果を検証します。

## クロスパネルバリュー

TimelineはLSP、Debug、AI、Tasksと最もよく連携し、何が起こったのかと次に何をすべきかの統一されたストーリーを提供します。
