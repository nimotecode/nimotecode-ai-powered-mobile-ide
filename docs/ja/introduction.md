# はじめに

NimoteCodeは、モバイルデバイスから本番レベルのコード修正とデリバリーを可能にするように設計された、モバイルファーストのIDEです。

## なぜNimoteCodeなのか？

開発者は常に移動中ですが、本番の問題は待ってくれません。NimoteCodeは、デスクトップクラスの開発ツールを携帯電話にもたらし、どこにいてもコードを修正、デバッグ、デプロイできるようにします。

## 主な特徴

- **SSHワークスペース** - リモートサーバーに直接接続
- **フルターミナル** - マルチタブと出力検索
- **Source Control** - 完全なGitワークフロー（Pro）
- **LSP & Debug** - 診断、コードアクション、ブレークポイントデバッグ（Pro）
- **AI統合** - AIチャットとエージェントオートメーション
- **Timeline診断** - イベントベースの根本原因分析
