# Features

Explore all the powerful features NimoteCode has to offer.

## Core Editor

### Syntax Highlighting

NimoteCode supports syntax highlighting for 50+ programming languages:

| Category | Languages |
|----------|-----------|
| **Web** | JavaScript, TypeScript, HTML, CSS, Vue, React, Svelte, Next.js |
| **Backend** | Python, Java, Go, Rust, C/C++, C#, PHP, Ruby, Node.js |
| **Mobile** | Swift, Kotlin, Dart, Flutter, React Native |
| **Data** | SQL, JSON, YAML, TOML, XML, CSV, Protobuf |
| **Scripting** | Bash, PowerShell, Python, Ruby, Perl |
| **Config** | Dockerfile, Kubernetes, Nginx, Apache, AWS CloudFormation |

### Code Folding

Collapse and expand code blocks for better navigation:

- Functions and methods
- Classes and structs
- Conditional blocks (if/else, switch/case)
- Loops (for, while, do-while)
- Comments and documentation
- Import statements and namespaces

### Editor Features

| Feature | Description |
|---------|-------------|
| **Auto-Save** | Changes saved automatically after 5 seconds |
| **Undo/Redo** | Multiple levels of undo history |
| **Cut/Copy/Paste** | Full clipboard support with keyboard shortcuts |
| **Select All** | Quick selection of all content |

### Large File Support

- Handle files up to **100,000 lines** efficiently
- Chunked reading for large files
- Automatic truncation warning for extremely large files
- Smooth scrolling performance with 60fps animations
- Memory-efficient file handling

### Save States

- **Dirty Flag**: Visual indicator for unsaved changes
- **Saving Indicator**: Progress indicator during save operations
- **Saved Confirmation**: Visual feedback when save completes
- **Auto-Save Timer**: Configurable auto-save delay

## SSH Remote Development

### Connection Management

| Feature | Details |
|---------|---------|
| **Authentication** | Password and SSH key support (RSA, ED25519, ECDSA) |
| **Timeout** | Configurable connection timeout (default 10 seconds) |
| **Auto-Reconnect** | Automatic reconnection with exponential backoff |
| **Status Display** | Real-time connection status with visual indicators |
| **Connection Pooling** | Multiple simultaneous connections |
| **Session Management** | Save and manage multiple server profiles |

### Remote File Browser

- **Tree View**: Visual file system navigation with expand/collapse
- **File Icons**: Type-specific icons for different file types
- **Path Navigation**: Breadcrumb navigation and path search
- **Directory Caching**: Performance optimization with intelligent caching
- **Refresh**: Manual and automatic directory refresh
- **Collapse All**: Quick collapse of all folders
- **Search**: Find files and directories by name
- **Filter**: Filter files by type or extension

### File Operations

| Operation | Description |
|-----------|-------------|
| **Create** | Create new files and directories |
| **Edit** | Real-time editing with auto-save |
| **Rename** | Rename files and directories |
| **Delete** | Delete files and directories with confirmation |
| **Copy** | Copy files and directories within server |
| **Move** | Move files between directories |
| **Upload** | Upload local files to remote server |
| **Download** | Download remote files to local device |
| **Permissions** | View and modify file permissions |
| **Symlinks** | View and manage symbolic links |

### Terminal Features

- **Shell Support**: Bash, Zsh, Fish, and custom shells
- **Tab Completion**: Intelligent command completion
- **Command History**: Persistent command history across sessions
- **Multi-Terminal**: Multiple terminal tabs in one session
- **Split Terminal**: Split terminal views for multiple commands
- **Copy/Paste**: Easy copy/paste with keyboard shortcuts
- **Search**: Search terminal output
- **Export**: Export terminal session to file
