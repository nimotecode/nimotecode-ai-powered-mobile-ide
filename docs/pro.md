# NimoteCode Pro

NimoteCode can be used as a core mobile workspace for local and SSH projects. Pro unlocks the higher-leverage workflows that turn that workspace into a complete mobile delivery environment.

## What Pro unlocks

| Area | Pro capability | Why it matters |
|------|----------------|----------------|
| AI Agent | Unlimited agent runs | Move beyond metered free automation for multi-step coding tasks. |
| Git | Write workflows and Git AI | Commit, push, branch, stash, review diffs, and use AI assistance from mobile. |
| Terminal | Multi-terminal workflows | Keep parallel shells for tests, logs, servers, and deployments. |
| Search | Unlimited remote content search | Search remote project files without the free usage meter. |
| LSP | Diagnostics and code actions | Navigate and fix code using language-server feedback. |
| Debug | Breakpoints, call stack, variables, watch | Investigate runtime problems from the app. |
| Timeline | AI root-cause and fix suggestions | Turn captured development events into diagnosis help. |
| Tasks | Advanced task execution | Run filtered task groups sequentially or in parallel. |
| Sync / Cache | Local/remote sync and cache flows | Move project content between local and remote workspaces with path checks. |

## Free capabilities

Free users can still start real work:

- Open local and SSH workspaces.
- Edit project files.
- Run baseline terminal sessions.
- Configure AI providers and use AI Chat.
- Use metered AI Agent runs.
- Use metered remote search.
- Create and run basic task workflows.
- View timeline information.

## Account and verification

Pro access is tied to account and subscription verification. The app supports restore-purchase flows and short offline grace behavior so temporary network issues do not immediately interrupt recently verified access.

See also: [Account and subscription](/docs/account-subscription).

## Best fit

Pro is best for developers who need to complete an entire mobile workflow, not just inspect code:

- Remote hotfixes that require Git writes.
- On-call debugging with LSP, debugger, and Timeline AI.
- Multi-terminal server workflows.
- Repeated AI Agent automation.
- Syncing or caching project files between local and remote environments.
