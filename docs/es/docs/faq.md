# FAQ

## ¿NimoteCode es solo para desarrollo remoto?

No. Puedes trabajar en modo Local Explorer o modo Remote Explorer (SSH).

## ¿Qué proveedores de AI están soportados?

Las plantillas incorporadas incluyen OpenAI, Anthropic, Gemini, Grok, DeepSeek, Zhipu, Moonshot, Ollama y puntos de conexión personalizados compatibles con OpenAI.

## ¿Qué puedo hacer en Source Control?

Puedes preparar/despreparar/descartar, confirmar, enviar/extraer/obtener, gestionar ramas/stash, ver historia y usar herramientas de análisis de Git AI.

## ¿Puedo ejecutar tareas de proyecto?

Sí. Usa el panel Tasks para crear y ejecutar tareas, luego inspecciona la salida en el terminal.

## ¿Para qué se usa Timeline?

Timeline te ayuda a inspeccionar eventos y trazas, incluyendo flujo de análisis causal para resolución de problemas.

## ¿Por qué está bloqueada una función?

Algunas capacidades están limitadas a Pro. El estado de suscripción es verificado por el flujo de acceso premium de la aplicación.
