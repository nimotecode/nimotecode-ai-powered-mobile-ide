# Terminal

El terminal es el centro de comandos para operaciones de compilación, prueba, despliegue e incidentes en NimoteCode.

## Capacidades principales

- Sesiones de terminal con múltiples pestañas
- Salida de comandos en tiempo real
- Búsqueda en salida para localizar registros rápidamente
- Atajos de comandos rápidos para acciones repetitivas
- Envía salida clave de terminal a AI para análisis
- Renombra y organiza pestañas para flujos de trabajo prolongados

## Viaje típico de comandos

1. Abre el terminal en el espacio de trabajo actual.
2. Ejecuta comandos build/test/deploy.
3. Busca errores y advertencias en la salida.
4. Escala registros complejos a AI o Timeline.

## Por qué importa

- Ciclo más rápido de comando a resultado en móvil
- Mejor visibilidad al manejar incidentes remotamente
- Mayor continuidad con editor, control de fuente y paneles AI
