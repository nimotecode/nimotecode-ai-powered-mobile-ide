# Configuración

La configuración te permite dar forma a NimoteCode en un entorno de desarrollo móvil estable para sesiones prolongadas.

## Cuenta y suscripción

- Inicio de sesión y gestión de identidad de cuenta
- Visibilidad de estado de suscripción y prueba
- Gestión de suscripción y entrada de restauración de compra

## Apariencia e idioma

- Control de modo de tema
- Selección de tema de IDE
- Cambio entre múltiples idiomas

## Experiencia del editor

- Ajuste de tamaño de fuente e interlineado
- Preferencias de familia de fuente
- Estilo de editor persistente entre sesiones

## Teclado y entrada

- Habilitación/deshabilitación de teclado suave
- Opciones de retroalimentación háptica y de sonido
- Alternancias de fila de números y fila de funciones
- Soporte de gestos y mapeo de teclas de función
- Tema de teclado y opciones de restablecimiento

## Acceso a políticas

La configuración incluye entradas directas a:

- [Eliminación de cuenta](/es/account-delete)
- [Política de privacidad](/es/privacy)
- [Términos de servicio](/es/terms)

## Primera configuración recomendada

1. Establece idioma y tema.
2. Ajusta la tipografía del editor.
3. Configura el comportamiento del teclado para tu dispositivo.
4. Confirma estado de suscripción y cuenta.
