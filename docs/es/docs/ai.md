# Asistente AI

NimoteCode incluye dos experiencias de AI: AI Chat para ayuda directa y AI Agent para ejecución de múltiples pasos.

## AI Chat

Usa AI Chat cuando quieras respuestas rápidas durante la codificación:

- Explica código desconocido
- Analiza errores de terminal
- Propone planes de refactorización
- Genera pasos siguientes más seguros antes del commit

## AI Agent

Usa AI Agent cuando el trabajo requiere múltiples acciones y verificaciones:

- Flujo de ejecución paso a paso
- Visibilidad de progreso y estado de ejecución
- Avisos de riesgo antes de acciones sensibles
- Modo automático o modo de confirmación según tu preferencia de control

## Flexibilidad de proveedores

NimoteCode soporta múltiples plantillas de proveedores:

- OpenAI
- Anthropic
- Gemini
- Grok
- DeepSeek
- Zhipu
- Moonshot
- Ollama
- Puntos de conexión personalizados compatibles con OpenAI

## Lo que puedes configurar

- Identidad del modelo y punto de conexión
- Comportamiento de muestreo y tamaño de salida
- Modo de respuesta en streaming
- Prompt del sistema y tamaño de contexto
- Uso de contexto de archivo para respuestas con conocimiento de la base de código

## Configuración de mejores prácticas

1. Mantén un modelo de nube principal y un modelo de respaldo.
2. Usa creatividad más baja para tareas de codificación deterministas.
3. Mantén el contexto de archivo activado para análisis a nivel de proyecto.
4. Revisa la salida de AI antes de ejecutar en producción.
