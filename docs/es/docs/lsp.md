# Panel LSP

El panel LSP te ayuda a detectar problemas antes y aplicar correcciones más rápido mientras trabajas en proyectos remotos.

## Valor principal

- Mostrar diagnósticos en un solo lugar
- Saltar directamente de problemas a líneas de código
- Aplicar acciones de código con menos pasos manuales
- Instalar y verificar soporte de servidor de lenguaje a través de flujo guiado

## Lo que puedes hacer

- Ver y filtrar errores y advertencias
- Abrir ubicación de problema instantáneamente en editor
- Actualizar y aplicar acciones de código disponibles
- Usar guía de instalador para servidores de lenguaje soportados
- Retroceder a flujos de corrección AI cuando acciones automáticas no estén disponibles

## Patrón de mejor uso

1. Ejecuta código y abre panel LSP.
2. Filtra primero a errores.
3. Aplica correcciones rápidas donde estén disponibles.
4. Revisa diagnósticos y repite.
5. Pasa casos complejos a Debug o AI.

## Notas Pro

LSP se proporciona como flujo de trabajo Pro y está optimizado para uso de espacio de trabajo SSH remoto activo.
