# Espacio de trabajo SSH

El espacio de trabajo SSH transforma NimoteCode en una consola de desarrollo remoto móvil.

## Lo que puedes hacer

- Guarda y reutiliza configuraciones de conexión a servidor
- Conecta usando contraseña o clave privada
- Desarrolla directamente en directorios de proyectos remotos
- Navega, edita, crea, renombra y elimina archivos remotamente
- Mantiene terminal y editor en el mismo contexto de espacio de trabajo

## Sesión remota típica

1. Selecciona un perfil SSH.
2. Conecta y abre la ruta raíz del espacio de trabajo.
3. Edita archivos y ejecuta comandos de terminal.
4. Completa verificación y entrega con Source Control y Debug/LSP.

## Prácticas recomendadas

- Divide perfiles por entorno
- Prioriza autenticación por clave para entornos críticos
- Especifica claramente la ruta raíz del espacio de trabajo para reducir riesgos de error
