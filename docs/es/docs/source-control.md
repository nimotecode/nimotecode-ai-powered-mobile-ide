# Source Control

Source Control trae flujos de trabajo Git completos a NimoteCode para que puedas revisar, confirmar y sincronizar cambios desde el móvil.

## Lo que puedes hacer

- Revisar archivos preparados, no preparados y en conflicto
- Preparar o descartar cambios en lote o por archivo
- Confirmar y sincronizar con pull/push/fetch
- Gestionar ramas y flujos de trabajo de stash
- Inspeccionar historia y diferencias de commit/archivo/línea

## Asistencia de Git AI

La AI incorporada puede ayudarte a:

- Generar mensajes de commit
- Explicar archivos modificados
- Resaltar riesgos potenciales de cambios

## Flujo típico de commit

1. Revisa diferencias de archivos.
2. Prepara cambios seleccionados.
3. Genera o escribe mensaje de commit.
4. Confirma y sincroniza.
5. Valida en rama objetivo.

## Por qué lo usan los equipos

- Entrega de código más rápida mientras estás en móvil
- Menor fricción para correcciones de emergencia
- Mejor calidad de cambios con revisión asistida por AI
