# Panel Debug

El panel Debug te da visibilidad de tiempo de ejecución móvil para que puedas diagnosticar y corregir problemas sin salir de tu flujo de trabajo.

## Lo que obtienes

- Controles de sesión: iniciar, pausar, continuar, paso, detener
- Gestión de puntos de interrupción y edición rápida
- Navegación por hilos y pila de llamadas
- Inspección de variables y observación
- Evaluación de expresiones durante estado pausado
- Flujo de salida integrado para contexto de depuración

## Por qué importa

- Tiempo más corto desde informe de error hasta causa raíz
- Menos cambios de contexto entre herramientas
- Mejor confianza antes de enviar correcciones

## Flujo típico de depuración móvil

1. Inicia sesión de depuración desde tu espacio de trabajo.
2. Alcanza punto de interrupción e inspecciona pila + variables.
3. Agrega expresiones de observación para estados inciertos.
4. Evalúa expresiones clave antes de cambios.
5. Aplica corrección y valida comportamiento.

## Integración con Timeline

Los eventos de depuración se reflejan en Timeline, por lo que la revisión post-incidente y el análisis de causa raíz permanecen conectados.
