# Panel Timeline

Timeline convierte señales de tiempo de ejecución en una narrativa práctica de resolución de problemas.

## Lo que entrega Timeline

- Vista de traza para análisis de causa y efecto
- Vista de lista de eventos para inspección cronológica
- Filtros de gravedad y fuente para reducir ámbito rápidamente
- Pistas de causa raíz y sugerencias de corrección
- Contadores de estado para conocimiento de tendencias de errores/advertencias

## Por qué importa

- Comprensión más rápida de incidentes
- Mejor alineación entre síntomas y causas raíz
- Contexto de traspaso claro para colaboración en equipo

## Flujo típico de resolución de problemas

1. Filtra a eventos de alta gravedad.
2. Abre trazas relevantes.
3. Revisa causas raíz probables.
4. Aplica sugerencias de corrección o escala a AI.
5. Valida resultado con nuevo flujo de eventos.

## Valor entre paneles

Timeline funciona mejor con LSP, Debug, AI y Tasks, dando una historia unificada de lo que pasó y qué hacer a continuación.
