# Panel Sync / Cache

Sync / Cache es un panel Pro para mover contenido de proyecto entre espacios de trabajo locales y remotos con control claro de dirección.

## Dos modos de transferencia

- Sync: de proyecto local a espacio de trabajo remoto
- Cache: de espacio de trabajo remoto a proyecto local

## Lo que obtienes

- Acción principal consciente del modo basada en contexto de espacio de trabajo
- Selección de ruta de destino restringida a límites de espacio de trabajo activo
- Historial de transferencia con registros de estado y ruta
- Visibilidad clara de estado antes y durante la ejecución

## Casos de uso típicos

- Enviar ediciones locales a entorno de ejecución remoto
- Extraer estado de proyecto remoto para revisión sin conexión
- Mantener una caché portátil de actualizaciones del lado del servidor

## Por qué importa

- Traspaso de entorno más rápido
- Mejor consistencia entre copias locales y remotas
- Flujo de trabajo de transferencia más seguro con dirección explícita y control de ruta
