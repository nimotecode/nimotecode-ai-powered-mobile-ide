# Introducción

NimoteCode es un IDE móvil primero diseñado para permitir correcciones y despliegues de código de nivel de producción directamente desde tu dispositivo móvil.

## ¿Por qué NimoteCode?

Los desarrolladores siempre están en movimiento, pero los problemas de producción no esperan. NimoteCode trae herramientas de desarrollo de clase de escritorio a tu teléfono, permitiéndote corregir, depurar y desplegar código dondequiera que estés.

## Funciones principales

- **Espacio de trabajo SSH**: conexión directa a servidores remotos
- **Terminal completo**: múltiples pestañas y búsqueda de salida
- **Source Control**: flujo de trabajo Git completo (Pro)
- **LSP y Debug**: diagnósticos, acciones de código, depuración con puntos de interrupción (Pro)
- **Integración AI**: chat AI y automatización con agente AI
- **Diagnóstico Timeline**: análisis de causa raíz basado en eventos
