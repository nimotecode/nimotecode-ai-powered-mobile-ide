  <h2 class="home-page-section-title">De acceder al servidor a entregar el cambio</h2>
  <p class="home-page-section-copy">NimoteCode está diseñado alrededor del camino real de trabajo: espacio de trabajo, editor, terminal, Git, IA, diagnóstico y sincronización.</p>
  <div class="home-page-grid">
    <div class="home-page-card">

      <p class="home-page-card-desc">Abre un proyecto local o conéctate por SSH con perfiles guardados, autenticación por contraseña o llave, y selección de rutas raíz.</p>
    </div>

      <div class="home-page-card-title"><span class="home-page-card-icon">02</span>Editar</div>
      <p class="home-page-card-desc">Usa un editor con pestañas, seguimiento del cursor, deshacer/rehacer, portapapeles, esquema y controles para teclado móvil.</p>
    </div>
    <div class="home-page-card">
      <div class="home-page-card-title"><span class="home-page-card-icon">03</span>Ejecutar</div>
      <p class="home-page-card-desc">Ejecuta comandos en terminal, busca salida, recupera sesiones tras reconexión y organiza terminales con Pro.</p>
    </div>
    <div class="home-page-card">
      <div class="home-page-card-title"><span class="home-page-card-icon">04</span>Buscar</div>
      <p class="home-page-card-desc">Busca contenido remoto con opciones de sensibilidad y regex. El uso gratis es limitado; Pro desbloquea búsqueda ilimitada.</p>
    </div>
    <div class="home-page-card">
      <div class="home-page-card-title"><span class="home-page-card-icon">05</span>Confirmar <span class="home-page-badge">Pro</span></div>
      <p class="home-page-card-desc">Revisa diffs, gestiona ramas, guarda cambios, confirma, pull, push, fetch y usa Git AI para entender cambios.</p>
    </div>
    <div class="home-page-card">
      <div class="home-page-card-title"><span class="home-page-card-icon">06</span>Diagnosticar <span class="home-page-badge">Herramientas Pro</span></div>
      <p class="home-page-card-desc">LSP, depurador, Timeline AI, Tasks y Sync/Cache para un control más profundo desde el móvil.</p>
