# Introduction

NimoteCode is a mobile-first IDE that lets you develop, debug, and ship from your phone or tablet.

## Why Teams Choose NimoteCode

- One workspace for coding, terminal, source control, AI, and troubleshooting
- Built for real SSH server workflows with auto-reconnection and terminal persistence
- Fast issue triage with LSP diagnostics, debug tools, and AI-powered timeline analysis with causal tracing
- Productive on mobile with keyboard preferences, editor typography, and focused panels

## What You Can Do

- Connect local or remote workspaces and edit project files instantly with heartbeat monitoring
- Run build/test/deploy commands in multi-tab terminal sessions with full persistence across reconnections
- Commit, sync, branch, stash, diff, and review changes in Source Control with Git AI assistance
- Use AI Chat and AI Agent with memory system (short/long-term memory, intelligent compression) to explain, refactor, and automate repetitive tasks
- Diagnose problems with LSP, Debug, Tasks, and Timeline panels with causal tracing and AI-powered root-cause analysis
- Sync project content between local and remote workspaces (Pro)
- Standardize recurring operations with Tasks panel

## Built for Real Delivery

NimoteCode is designed for practical delivery scenarios:

1. Remote hotfix while away from laptop with stable SSH connections
2. On-call incident triage with logs + timeline traces and causal analysis
3. Daily commit workflow with in-app source control and Git AI
4. AI-assisted coding and repetitive workflow automation with memory persistence
5. Multi-terminal sessions that survive network interruptions
