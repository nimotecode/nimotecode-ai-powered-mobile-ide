# Удаление аккаунта

Ниже описано, как удалить ваш аккаунт и все данные.

## Удаление в приложении

1. Откройте приложение NimoteCode
2. Перейдите в **Настройки** → **Аккаунт**
3. Нажмите **Удалить аккаунт**
4. Следуйте инструкциям подтверждения

## Удаление данных

При удалении аккаунта следующее будет удалено навсегда:
- Информация об аккаунте
- Профили серверов
- Настройки приложения
- История чатов
- Данные AI-агента

## Удаление по запросу

Если у вас нет доступа к приложению, свяжитесь с службой поддержки. После подтверждения мы обработаем удаление вручную.
