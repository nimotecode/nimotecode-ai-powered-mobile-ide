# Политика конфиденциальности

Последнее обновление: 2024 год

## Введение

NimoteCode обязуется уважать и защищать вашу конфиденциальность. Эта политика конфиденциальности объясняет, как мы собираем, используем и делимся информацией.

## Собираемая информация

- **Информация об аккаунте**: адрес электронной почты, учетные данные
- **Данные использования**: как вы взаимодействуете с приложением и используете функции
- **Данные подключения**: профили SSH-серверов (сохраняются на устройстве)
- **Диагностические данные**: отчеты о сбоях и данные о производительности (опционально)

## Как мы используем данные

- Для предоставления и поддержания услуг
- Для улучшения взаимодействия с приложением
- Для предоставления поддержки
- Для защиты безопасности

## Обмен данными

Мы не продаем и не делимся вашими данными с третьими лицами, за исключением следующих случаев:
- С вашего согласия
- По требованию закона
- Для защиты наших прав

## Ваши права

- Доступ к данным
- Исправление данных
- Удаление данных
- Экспорт данных
