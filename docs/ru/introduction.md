# Введение

NimoteCode — это мобильное IDE, разработанное для обеспечения исправления и развертывания кода производственного уровня прямо с вашего мобильного устройства.

## Почему NimoteCode?

Разработчики всегда в движении, но проблемы в производстве не ждут. NimoteCode приносит инструменты разработки класса настольных приложений на ваш телефон, позволяя исправлять, отлаживать и развертывать код где бы вы ни находились.

## Основные функции

- **SSH-рабочая область** — прямое подключение к удаленным серверам
- **Полный терминал** — множество вкладок и поиск по выводу
- **Source Control** — полный рабочий процесс Git (Pro)
- **LSP и Debug** — диагностика, действия с кодом, отладка с точками останова (Pro)
- **Интеграция с AI** — AI-чат и автоматизация с помощью AI-агента
- **Диагностика Timeline** — анализ корневых причин на основе событий
