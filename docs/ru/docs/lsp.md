# Панель LSP

Панель LSP помогает вам обнаруживать проблемы раньше и применять исправления быстрее при работе над удаленными проектами.

## Основная ценность

- Отображение диагностики в одном месте
- Переход напрямую из проблем в исходные строки
- Применение действий с кодом с меньшим количеством ручных шагов
- Установка и проверка поддержки языкового сервера через направленный поток

## Что вы можете сделать

- Просматривать и фильтровать ошибки и предупреждения
- Мгновенно открывать местоположение проблемы в редакторе
- Обновлять и применять доступные действия с кодом
- Использовать руководство установщика для поддерживаемых языковых серверов
- Возврат к потокам исправления AI, когда автоматические действия недоступны

## Лучший шаблон использования

1. Запустите код и откройте панель LSP.
2. Сначала фильтруйте до ошибок.
3. Применяйте быстрые исправления, где доступно.
4. Перепроверяйте диагностику и повторяйте.
5. Передавайте сложные случаи в Debug или AI.

## Примечания Pro

LSP предоставляется как рабочий процесс Pro и оптимизирован для использования активной удаленной SSH-рабочей области.
