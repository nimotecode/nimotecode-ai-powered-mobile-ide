# Source Control

Source Control приносит полные рабочие процессы Git в NimoteCode, чтобы вы могли проверять, фиксировать и синхронизировать изменения с мобильного устройства.

## Что вы можете сделать

- Просматривать подготовленные, не подготовленные и конфликтующие файлы
- Подготовлять или отменять изменения пакетом или по отдельным файлам
- Фиксировать и синхронизировать с pull/push/fetch
- Управлять ветками и рабочими процессами stash
- Просматривать историю и различия коммитов/файлов/строк

## Помощь Git AI

Встроенный AI может помочь вам:

- Генерировать сообщения коммитов
- Объяснять измененные файлы
- Подчеркивать потенциальные риски изменений

## Типичный поток коммитов

1. Просмотрите различия файлов.
2. Подготовьте выбранные изменения.
3. Сгенерируйте или напишите сообщение коммита.
4. Зафиксируйте и синхронизируйте.
5. Проверьте в целевой ветке.

## Почему команды используют это

- Более быстрая доставка кода при работе с мобильного устройства
- Меньше трения для экстренных исправлений
- Лучшее качество изменений с проверкой с помощью AI
