# Конфигурация

Эта страница помогает вам настроить NimoteCode для стабильного ежедневного использования с упором на AI, SSH и рабочие процессы задач.

## Стратегия конфигурации AI

Создайте как минимум два профиля:

- Основной профиль для ежедневного кодирования
- Резервный профиль для отказа или контроля затрат

Настройте каждый профиль под ваш стиль использования:

- Более низкая случайность для точных задач кодирования
- Более высокий контекст для рассуждений о больших проектах
- Потоковая передача включена для более быстрого воспринимаемого ответа

## Контрольный список настройки провайдера AI

- Добавьте провайдера, API-ключ, конечную точку и модель
- Проверьте ответы в AI Chat
- Подтвердите, что AI Agent может выполнять с вашим выбранным профилем
- Поддерживайте запросы и лимиты согласованными с целями затрат/производительности
- API-ключи хранятся через безопасное хранилище платформы (например, хранилище на основе iOS Keychain / Android Keystore)

## Стратегия конфигурации SSH

Для каждой серверной среды поддерживайте выделенный профиль:

- Имя среды (например: Staging, Production)
- Хост, порт, пользователь и метод аутентификации
- Стабильный корневой путь рабочего пространства для предсказуемой навигации

Рекомендуется:

- Используйте аутентификацию по ключу для производственных серверов
- Разделяйте среды высокого и низкого риска
- Проверяйте права доступа перед запуском задач
- Если используется аутентификация по паролю, рассматривайте ее как только для сессии: пароль используется в памяти и не сохраняется в локальном хранилище профиля SSH

## Стратегия конфигурации Tasks

Проектируйте задачи вокруг повторяющихся результатов:

- Сборка и тестирование
- Линтинг и форматирование
- Перезапуск службы и проверки работоспособности
- Быстрые доступы к проверке журналов

Советы:

- Используйте четкие имена задач по конвенции команды
- Группируйте связанные задачи для быстрого фильтрования
- Поддерживайте четко помеченными команды высокого риска

## Базовый уровень готовности к запуску

Перед ежедневным использованием подтвердите:

1. Профиль AI работает в Chat и Agent
2. Профиль SSH может подключаться и открывать рабочее пространство
3. Хотя бы одна задача сборки/тестирования успешно запускается
4. Source Control может завершать фиксацию и синхронизацию
