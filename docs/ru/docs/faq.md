# FAQ

## NimoteCode только для удаленной разработки?

Нет. Вы можете работать в режиме Local Explorer или режиме Remote Explorer (SSH).

## Какие провайдеры AI поддерживаются?

Встроенные шаблоны включают OpenAI, Anthropic, Gemini, Grok, DeepSeek, Zhipu, Moonshot, Ollama и пользовательские конечные точки, совместимые с OpenAI.

## Что я могу сделать в Source Control?

Вы можете подготовить/отменить подготовку/отменить, фиксировать, отправлять/извлекать/получать, управлять ветками/stash, просматривать историю и использовать инструменты анализа Git AI.

## Могу ли я запускать задачи проекта?

Да. Используйте панель Tasks для создания и запуска задач, затем проверяйте вывод в терминале.

## Для чего используется Timeline?

Timeline помогает вам проверять события и трассировки, включая поток анализа причин для устранения неполадок.

## Почему функция заблокирована?

Некоторые возможности ограничены Pro. Статус подписки проверяется потоком премиум-доступа приложения.
