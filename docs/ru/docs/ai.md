# AI-ассистент

NimoteCode включает два опыта с AI: AI Chat для прямой помощи и AI Agent для многошагового выполнения.

## AI Chat

Используйте AI Chat, когда вам нужны быстрые ответы во время кодирования:

- Объяснение незнакомого кода
- Анализ ошибок терминала
- Предложение планов рефакторинга
- Генерация более безопасных следующих шагов перед коммитом

## AI Agent

Используйте AI Agent, когда работа требует нескольких действий и проверок:

- Пошаговый поток выполнения
- Видимость прогресса и статус выполнения
- Предупреждения о рисках перед конфиденциальными действиями
- Автоматический режим или режим подтверждения в зависимости от ваших предпочтений контроля

## Гибкость провайдеров

NimoteCode поддерживает несколько шаблонов провайдеров:

- OpenAI
- Anthropic
- Gemini
- Grok
- DeepSeek
- Zhipu
- Moonshot
- Ollama
- Пользовательские конечные точки, совместимые с OpenAI

## Что вы можете настроить

- Идентификатор модели и конечная точка
- Поведение выборки и размер вывода
- Режим потокового ответа
- Системный запрос и размер контекста
- Использование контекста файла для ответов с учетом кодовой базы

## Настройка по лучшим практикам

1. Поддерживайте одну основную облачную модель и одну резервную.
2. Используйте более низкую креативность для детерминированных задач кодирования.
3. Поддерживайте контекст файла включенным для анализа на уровне проекта.
4. Проверяйте вывод AI перед запуском в производстве.
