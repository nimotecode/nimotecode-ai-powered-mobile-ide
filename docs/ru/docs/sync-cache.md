# Панель Sync / Cache

Sync / Cache — это панель Pro для перемещения содержимого проекта между локальными и удаленными рабочими пространствами с четким контролем направления.

## Два режима передачи

- Sync: из локального проекта в удаленное рабочее пространство
- Cache: из удаленного рабочего пространства в локальный проект

## Что вы получаете

- Основное действие с учетом режима на основе контекста рабочего пространства
- Выбор целевого пути, ограниченного границами активного рабочего пространства
- История передачи с записями состояния и пути
- Ясная видимость состояния до и во время выполнения

## Типичные варианты использования

- Отправка локальных правок в удаленную среду выполнения
- Извлечение состояния удаленного проекта для автономного просмотра
- Поддержание переносимого кэша обновлений на стороне сервера

## Почему это важно

- Более быстрая передача среды
- Лучшая согласованность между локальными и удаленными копиями
- Более безопасный рабочий процесс передачи с явным контролем направления и пути
