# 개인정보 처리방침

최종 업데이트: 2024년

## 서론

NimoteCode는 귀하의 개인정보를 존중하고 보호하는 것을 약속합니다. 이 개인정보 처리방침은 당사가 정보를 수집, 사용, 공유하는 방법을 설명합니다.

## 수집하는 정보

- **계정 정보**: 이메일 주소, 인증 정보
- **사용 데이터**: 앱 작업과 기능 사용 방법
- **연결 데이터**: SSH 서버 프로필 (기기에 저장)
- **진단 데이터**: 크래시 보고서와 성능 데이터 (옵트인)

## 데이터 사용 방법

- 서비스 제공 및 유지
- 앱 경험 개선
- 지원 제공
- 보안 보호

## 데이터 공유

당사는 귀하의 데이터를 제3자와 판매하거나 공유하지 않습니다. 다음 경우를 제외합니다:
- 귀하의 동의가 있는 경우
- 법률로 의무화된 경우
- 당사의 권리를 보호하기 위해

## 귀하의 권리

- 데이터 액세스
- 데이터 수정
- 데이터 삭제
- 데이터 내보내기

## 문의

개인정보에 관한 질문이 있으시면 문의해 주세요.
