# 소개

NimoteCode는 모바일 기기에서 프로덕션 수준의 코드 수정과 배포를 가능하게 하도록 설계된 모바일 우선 IDE입니다.

## 왜 NimoteCode인가?

개발자는 항상 이동 중이지만, 프로덕션 문제는 기다려주지 않습니다. NimoteCode는 데스크톱 클래스 개발 도구를 휴대폰으로 가져와, 어디에 있든 코드를 수정, 디버그, 배포할 수 있게 합니다.

## 주요 기능

- **SSH 워크스페이스** - 원격 서버에 직접 연결
- **전체 터미널** - 멀티 탭과 출력 검색
- **Source Control** - 완전한 Git 워크플로우 (Pro)
- **LSP & Debug** - 진단, 코드 작업, 중단점 디버그 (Pro)
- **AI 통합** - AI 채팅과 에이전트 자동화
- **Timeline 진단** - 이벤트 기반 근본 원인 분석
