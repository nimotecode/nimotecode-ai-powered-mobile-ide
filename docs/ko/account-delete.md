# 계정 삭제

계정과 모든 데이터를 삭제하는 방법은 다음과 같습니다.

## 앱 내에서 삭제

1. NimoteCode 앱을 엽니다
2. **설정** → **계정**으로 이동합니다
3. **계정 삭제**를 탭합니다
4. 확인 지침을 따릅니다

## 데이터 삭제

계정을 삭제하면 다음이 영구적으로 삭제됩니다:
- 계정 정보
- 서버 프로필
- 앱 설정
- 채팅 히스토리
- AI 에이전트 데이터

## 요청에 의한 삭제

앱에 액세스할 수 없는 경우, 지원팀에 문의하세요. 확인 후 수동으로 삭제를 처리합니다.
