# AI 어시스턴트

NimoteCode에는 두 가지 AI 경험인 직접 지원을 위한 AI 채팅과 다단계 실행을 위한 AI 에이전트가 포함되어 있습니다.

## AI 채팅

코딩 중에 빠른 답변이 필요할 때 AI 채팅을 사용합니다:

- 익숙하지 않은 코드 설명
- 터미널 오류 분석
- 리팩토링 계획 제안
- 커밋 전에 더 안전한 다음 단계 생성

## AI 에이전트

작업에 여러 작업과 확인이 필요할 때 AI 에이전트를 사용합니다:

- 단계별 실행 흐름
- 진행 가시성과 실행 상태
- 민감한 작업 전의 위험 프롬프트
- 제어 선호도에 따른 자동 모드 또는 확인 모드

## 프로바이더 유연성

NimoteCode는 여러 프로바이더 템플릿을 지원합니다:

- OpenAI
- Anthropic
- Gemini
- Grok
- DeepSeek
- Zhipu
- Moonshot
- Ollama
- 사용자 정의 OpenAI 호환 엔드포인트

## 구성할 수 있는 것

- 모델 ID와 엔드포인트
- 샘플링 동작과 출력 크기
- 스트리밍 응답 모드
- 시스템 프롬프트와 컨텍스트 크기
- 코드베이스 인식 응답을 위한 파일 컨텍스트 사용

## 모범 사례 설정

1. 기본 클라우드 모델 하나와 백업 모델 하나를 유지합니다.
2. 결정론적 코드 작업에는 낮은 창의성을 사용합니다.
3. 프로젝트 수준 분석을 위해 파일 컨텍스트를 켭니다.
4. 프로덕션에서 실행하기 전에 AI 출력을 검토합니다.
