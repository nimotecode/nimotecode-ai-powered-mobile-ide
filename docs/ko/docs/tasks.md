# Tasks 패널

Tasks 패널은 수동 입력을 적게 하면서 원격 프로젝트에서 반복 가능한 명령 워크플로우를 실행하는 데 도움이 됩니다.

## 할 수 있는 것

- 그룹별로 작업 정리
- 작업 카탈로그 검색 및 필터링
- 도킹된 SSH 터미널 컨텍스트에서 작업 실행
- 시작부터 완료까지 실행 상태 추적
- 작업 정의 추가, 편집, 관리

## 팀이 사용하는 이유

- 표준화된 일일 운영
- 루틴 명령의 더 빠른 실행
- 반복 워크플로우에서의 인적 오류 감소

## 전형적인 워크플로우

1. 작업 그룹으로 필터링합니다.
2. 패널에서 작업을 실행합니다.
3. 출력과 상태를 모니터링합니다.
4. 필요한 경우 작업 설정을 조정합니다.

## 이상적인 작업 카테고리

- 빌드 및 테스트
- 린트 및 포맷
- 배포 및 재시작
- 진단 및 로그 확인
