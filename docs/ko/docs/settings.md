# 설정

설정을 통해 NimoteCode를 안정적인 장시간 세션 모바일 개발 환경으로 만들 수 있습니다.

## 계정 및 구독

- 로그인 및 계정 ID 관리
- 구독 상태 및 평가판 가시성
- 구독 관리 및 구매 복원 항목

## 모양 및 언어

- 테마 모드 제어
- IDE 테마 선택
- 다국어 전환

## 편집기 경험

- 글꼴 크기 및 줄 간격 조정
- 글꼴 패밀리 기본 설정
- 세션 간 영구적인 편집기 스타일

## 키보드 및 입력

- 소프트 키보드 켜기/끄기
- 햅틱 및 사운드 피드백 옵션
- 숫자 행 및 기능 행 토글
- 제스처 지원 및 기능 키 매핑
- 키보드 테마 및 재설정 옵션

## 정책 액세스

설정에는 다음에 대한 직접 항목이 포함됩니다:

- [계정 삭제](/ko/account-delete)
- [개인정보 처리방침](/ko/privacy)
- [이용 약관](/ko/terms)

## 권장되는 첫 설정

1. 언어와 테마를 설정합니다.
2. 편집기 조판을 조정합니다.
3. 기기에 맞게 키보드 동작을 구성합니다.
4. 구독과 계정 상태를 확인합니다.
