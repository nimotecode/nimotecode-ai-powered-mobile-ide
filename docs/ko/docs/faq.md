# FAQ

## NimoteCode는 원격 개발만을 위한 것인가요?

아닙니다. Local Explorer 모드 또는 Remote Explorer (SSH) 모드에서 작업할 수 있습니다.

## 어떤 AI 프로바이더가 지원되나요?

내장 템플릿에는 OpenAI, Anthropic, Gemini, Grok, DeepSeek, Zhipu, Moonshot, Ollama, 사용자 정의 OpenAI 호환 엔드포인트가 포함됩니다.

## Source Control에서 무엇을 할 수 있나요?

스테이징/언스테이징/취소, 커밋, 푸시/풀/페치, 브랜치/stash 관리, 기록 보기, Git AI 분석 도구 사용을 할 수 있습니다.

## 프로젝트 작업을 실행할 수 있나요?

네. Tasks 패널을 사용하여 작업을 만들고 실행한 다음 터미널에서 출력을 검사할 수 있습니다.

## Timeline은 무엇에 사용되나요?

Timeline은 트러블슈팅을 위한 인과 분석 흐름을 포함하여 이벤트와 추적을 검사하는 데 도움이 됩니다.

## 기능이 잠겨 있는 이유는 무엇인가요?

일부 기능은 Pro로 제한됩니다. 구독 상태는 앱의 프리미엄 액세스 흐름에 의해 확인됩니다.
