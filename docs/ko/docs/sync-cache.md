# Sync / Cache 패널

Sync / Cache는 명확한 방향 제어로 로컬과 원격 워크스페이스 간에 프로젝트 콘텐츠를 이동하기 위한 Pro 패널입니다.

## 두 가지 전송 모드

- Sync: 로컬 프로젝트에서 원격 워크스페이스로
- Cache: 원격 워크스페이스에서 로컬 프로젝트로

## 얻을 수 있는 것

- 워크스페이스 컨텍스트 기반의 모드 인식 기본 작업
- 활성 워크스페이스 경계로 제한된 대상 경로 선택
- 상태 및 경로 기록이 포함된 전송 기록
- 실행 전과 실행 중의 명확한 상태 가시성

## 전형적인 사용 사례

- 로컬 편집을 원격 런타임 환경으로 푸시
- 오프라인 검토를 위해 원격 프로젝트 상태 풀
- 서버 측 업데이트의 포터블 캐시 유지

## 중요한 이유

- 더 빠른 환경 인계
- 로컬과 원격 사본 간의 더 나은 일관성
- 명시적 방향과 경로 제어로 더 안전한 전송 워크플로우
