# 터미널

터미널은 NimoteCode에서 빌드, 테스트, 배포, 인시던트 작업의 명령 센터입니다.

## 핵심 기능

- 멀티 탭 터미널 세션
- 실시간 명령 출력
- 빠른 로그 타겟팅을 위한 출력 검색
- 반복 작업을 위한 빠른 명령 단축키
- 분석을 위해 주요 터미널 출력을 AI로 보내기
- 긴 워크플로우를 위한 탭 이름 변경과 정리

## 전형적인 명령 여정

1. 현재 워크스페이스에서 터미널을 엽니다.
2. build/test/deploy 명령을 실행합니다.
3. 오류와 경고의 출력을 검색합니다.
4. 복잡한 로그를 AI 또는 Timeline으로 에스컬레이션합니다.

## 중요한 이유

- 모바일에서 명령에서 결과까지의 루프가 더 빠릅니다
- 원격에서 인시던트를 처리할 때 가시성이 향상됩니다
- 편집기, 소스 제어, AI 패널과의 연속성이 강화됩니다
