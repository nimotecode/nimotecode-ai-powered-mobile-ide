# SSH 워크스페이스

SSH 워크스페이스는 NimoteCode를 모바일 원격 개발 콘솔로 변환합니다.

## 할 수 있는 것

- 서버 연결 설정을 저장하고 재사용합니다
- 비밀번호 또는 개인 키 방식으로 연결합니다
- 원격 프로젝트 디렉토리에서 직접 개발합니다
- 원격에서 파일을 탐색, 편집, 생성, 이름 변경, 삭제합니다
- 터미널과 편집기를 동일한 워크스페이스 컨텍스트로 유지합니다

## 전형적인 원격 세션

1. SSH 프로필을 선택합니다.
2. 연결하고 워크스페이스 루트 경로를 엽니다.
3. 파일을 편집하고 터미널 명령을 실행합니다.
4. Source Control과 Debug/LSP로 검증과 배포를 완료합니다.

## 권장 사례

- 환경별로 프로필을 분할합니다
- 중요한 환경에서는 개인 키 인증을 우선 사용합니다
- 워크스페이스 루트 경로를 명확히 하여 오작동 위험을 줄입니다
