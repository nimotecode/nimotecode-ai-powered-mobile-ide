# LSP 패널

LSP 패널은 원격 프로젝트에서 작업하는 동안 문제를 조기에 발견하고 수정을 더 빨리 적용하는 데 도움이 됩니다.

## 핵심 가치

- 한 곳에 진단 표시
- 문제에서 소스 라인으로 직접 이동
- 수동 단계를 적게 하여 코드 작업 적용
- 안내 흐름을 통해 언어 서버 지원 설치 및 검증

## 할 수 있는 것

- 오류와 경고를 보고 필터링
- 편집기에서 문제 위치 즉시 열기
- 사용 가능한 코드 작업 새로 고침 및 적용
- 지원되는 언어 서버의 설치 안내 사용
- 자동 작업을 사용할 수 없을 때 AI 수정 흐름으로 폴백

## 모범 사용 패턴

1. 코드를 실행하고 LSP 패널을 엽니다.
2. 먼저 오류로 필터링합니다.
3. 가능한 경우 빠른 수정을 적용합니다.
4. 진단을 다시 확인하고 반복합니다.
5. 복잡한 사례를 Debug 또는 AI로 넘깁니다.

## Pro 노트

LSP는 Pro 워크플로우로 제공되며 활성 SSH 원격 워크스페이스 사용에 최적화되어 있습니다.
