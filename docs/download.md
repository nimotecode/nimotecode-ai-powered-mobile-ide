## Installation Guide

### Android

1. **From Play Store:**
   - Open Google Play Store
   - Search for "NimoteCode"
   - Tap "Install"
   - Open the app

2. **From APK:**
   - Download the APK from GitHub releases
   - Enable "Install from unknown sources" in settings
   - Open the APK file
   - Tap "Install"
   - Open the app

### iOS

1. Open the App Store
2. Search for "NimoteCode"
3. Tap "Get"
4. Open the app

## First Launch

When you first open NimoteCode:

1. **Sign In**: Create an account or sign in with Google/Apple
2. **Choose a Plan**: Select Free or Professional
3. **Add Connection**: Set up your first SSH or GitHub connection
4. **Start Coding**: Open a file and start editing!
