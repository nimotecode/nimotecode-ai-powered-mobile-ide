# Tasks Panel

Tasks Panel helps you run repeatable command workflows in remote projects with less manual typing.

## What You Can Do

- Organize tasks by groups
- Search and filter task catalog
- Run tasks in docked SSH terminal context
- Track run states from start to completion
- Add, edit, and manage task definitions

## Why Teams Use It

- Standardized daily operations
- Faster execution of routine commands
- Lower human error on repetitive workflows

## Typical Workflow

1. Filter to a task group.
2. Run task from panel.
3. Watch output and status.
4. Adjust task settings if needed.

## Ideal Task Categories

- Build and test
- Lint and format
- Deploy and restart
- Diagnostic and log checks
